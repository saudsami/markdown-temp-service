Markdown Temporary Hosting Service

This service provides temporary markdown file hosting for AI chat integrations.

API Endpoints

  • POST /api/temp-markdown/create - Create temporary markdown file
  • GET /api/temp-markdown/{id} - Retrieve markdown file
  • GET /api/health - Health check

Example Usage

// Create temporary markdown
fetch('/api/temp-markdown/create', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    content: '# My Markdown\n\nContent here...',
    title: 'My Document',
    expiresInHours: 24
  })
})
.then(r => r.json())
.then(data => console.log('URL:', data.url));
